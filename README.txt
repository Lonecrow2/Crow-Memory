# Crow Memory

**Divirta-se com Crow Memory**, um jogo da memória temático de corvos, feito para desktop e celular.

## Descrição

Crow Memory é um jogo da memória com temática de corvos, composto por 8 personagens únicos e 16 cartas ao todo.  
Ao iniciar, as cartas são embaralhadas utilizando o algoritmo Fisher-Yates para garantir uma distribuição aleatória.  
O cronômetro começa no primeiro clique, e cada par corretamente encontrado adiciona um ponto à pontuação do jogador.  
Ao finalizar a partida, basta clicar em “Iniciar/Reiniciar” para embaralhar novamente e iniciar uma nova rodada.

**Estilo de jogo:**  
Jogo de memória interativo estilo *point-and-click*, com mecânica de matching visual.

## Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript  

## Compatibilidade

- Funciona em navegadores desktop e mobile  
- Responsivo para diferentes resoluções  
- Ideal para tela vertical (mobile) ou horizontal (desktop)
